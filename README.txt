FLAWLESS GRAPHICS Website
=========================
A static multi-page website for managing employee records, attendance, and payroll.

Pages:
- Home (index.html)
- Employee Login (employee.html)
- HR Dashboard (hr.html)
- Services (services.html)
- Contact (contact.html)

Features:
- Responsive design
- Automatic light/dark mode
- Demo login buttons
- HR Dashboard with sample data
- Functional contact form (ready to connect to API)
- Placeholder logo and contact info

To customize:
- Replace text inside [square brackets] with your real details.
- Replace placeholder icons in /assets.
- To make contact form work, replace the fetch() URL in contact.html with your backend endpoint.
